# TooDoo

A simple **Todo application** built with **Next.js** and **Chakra UI**.  

Manage tasks, track priorities, and organize your workflow.  

---

## Features

- Add, update, and remove todos  
- Assign priorities and assignees  
- Persist todos in `localStorage`  
- Responsive design with Chakra UI  

---

## Tech Stack

- [Next.js](https://nextjs.org/)  
- [React](https://reactjs.org/)  
- [Chakra UI](https://chakra-ui.com/)  
- TypeScript  

---

## Getting Started

### Prerequisites

- Node.js >= 20.x  
- npm or yarn  

---

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/toodoo.git
cd toodoo

```

2. Install dependencies:

```bash
npm install
# or
yarn install

```
